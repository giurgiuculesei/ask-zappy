Medium

1) If a sequence has a1 = 5 and common difference d = 3, find a20.

2) Solve for x: 2·3x = 54.

3) Find the inverse of f(x) = (2x − 3)/(x + 4).

Mark Scheme

1) an = a1 + (n−1)d ⇒ a20 = 5 + 19·3 = 62.

2) 2·3x = 54 ⇒ 3x = 27 ⇒ x = 3.

3) y = (2x − 3)/(x + 4) ⇒ x = (3 + 4y)/(2 − y) ⇒ f−1(x) = (3 + 4x)/(2 − x).
